# Quotes and Status App

Quotes and Status is a quotes app that allows users to browse through various categories and select quotes based on their needs or mood. The app is built using Kotlin and 
Java programming languages and integrates Firebase for seamless functionality. This README.md file provides an overview of the app's features and instructions on how to use it.


## Features

* **Browse Categories:** Users can explore different categories to find quotes that match their needs or mood.
* **Vast Collection of Quotes**: Each category contains a wide range of quotes for users to choose from.
* **Copy Quotes**: Users have the option to copy the selected quote and use it anywhere they want.
* **Save as Image:** Quotes can be saved as images in the device's gallery for later use or sharing.
* **Change Background:** Tapping on a quote allows users to change the background of the quote.
* **Like Quotes:** Users can like their favorite quotes to keep track of them.
* **Offline and Online Support:** The app works seamlessly in both offline and online modes.


## How to Use
1.  **Browse Categories:** Open the app and browse through the available categories to find quotes that resonate with your needs or mood.
2.  **Select a Quote:** Once you have chosen a category, browse through the quotes in that category.
3.  **Copy Quote:** If you find a quote you like, tap on it to view it in detail. From there, you can copy the quote by selecting the copy option.
4.  **Save as Image:** To save a quote as an image, tap on the quote to view it in detail and select the option to save it as an image. The image will be saved in your device's gallery.
5.  **Change Background:** While viewing a quote in detail, tap on the quote to change its background. Choose the background that suits your preference.
6.  **Like Quotes:** If you come across a quote that you particularly like, tap the like button to mark it as a favorite.
7.  **Offline Usage:** The app is designed to work offline as well. You can access previously viewed quotes and use the app's features without an internet connection.


## Technologies Used

The Quotes and Status app is built using the following technologies:

* Kotlin: The primary programming language used for developing the app.
* Java: Some components of the app are written in Java.
* Firebase: Integration with Firebase provides various functionalities such as data storage, offline support, and user authentication.


## Getting Started
To set up the Quotes and Status app locally and start working on it, follow these steps:

1. Clone the repository: git clone **https://github.com/samiali12/quotes-and-status-app.git**
2. Open the project in your preferred IDE or code editor.
3. Ensure you have the necessary dependencies installed.
4. Set up Firebase integration by following the Firebase documentation.
5. Build and run the app on an emulator or physical device.
6. Start exploring the quotes and customizing the app according to your requirements.
